6 Bangun Mathematical Model

Model ini menjadi dasar simulasi Monte Carlo untuk risiko yang dipilih. Distribusi random yang digunakan: Normal.
Penanda Bangun Mathematical Model: bagian ini adalah titik aplikasi mengubah data historis menjadi model matematis. Model tersebut kemudian dipakai untuk menjalankan Monte Carlo Simulation, menghitung distribusi output, probabilitas target, gap, potential loss, dan VaR.
Tahap Model Rumus / Logika Output
Bangun Mathematical Model model = actual_ytd + sum(forecast_bulanan_random) Data historis diubah menjadi mesin forecast berbasis simulasi.
Growth historis bulanan growth_t = (nilai_t / nilai_t-1) - 1 Mean growth dan standar deviasi growth dari data historis.
Forecast bulanan simulasi forecast_t = max(forecast_t-1 * (1 + random_growth), 0) {{ risk_prediction_detail.simulation_count|default:"10.000" }} skenario nilai bulanan ke depan.
Monte Carlo Simulation Simulasi diulang:
for i in range(n_simulations)		 Jumlah simulasi hasil terakhir: {{ risk_prediction_detail.simulation_count|default:"10.000" }} kali.
Distribusi random Distribusi Normal:
random_growth = Normal(mean_growth, std_growth)		 Variasi naik/turun mengikuti pola volatilitas historis.
Total simulasi tahunan simulated_total = actual_ytd + sum(forecast_bulanan) Total forecast sampai akhir tahun untuk setiap simulasi.
Forecast utama forecast_total = P50(simulated_total) Baseline / median, bukan rata-rata tunggal.
STDEV F Excel STDEV_F = P50(forecast_bulanan) - P15,8655254(forecast_bulanan) Dipakai untuk tabel forecast bulanan seperti contoh Excel.
Status target simulated_total >= target_rkap Target tercapai jika simulasi total minimal sama dengan target.
Gap dan potential loss gap = max(target_rkap - forecast_total, 0)
potential_loss = gap * harga_jual_rata_rata		 Estimasi selisih target dan dampak finansial.
VaR 95% Hitung VaR:
VaR95 = max(target_rkap - P5(simulated_total), 0)		 Estimasi downside pada skenario 5% terburuk.
Keputusan mitigasi prob_tidak_tercapai >= risk_appetite Jika melewati appetite, risiko perlu mitigasi.
